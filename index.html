Pokemon Api Project

This project utilizes the PokéAPI to dynamically generate cards for the original 150 Pokémon. Each card displays the Pokémon's image, name, number (from 1 to 150), and type. The card's background color changes based on the Pokémon's type, providing a visually distinct appearance for each card. The project showcases the integration of API data retrieval, DOM manipulation, and responsive design techniques to create an interactive and visually appealing Pokémon display.

Leno App Project

This project is a responsive marketing website for the Leno App, developed using HTML, CSS, and JavaScript. It features a clean, modern design that emphasizes user engagement and conversion. The website layout is fully responsive, ensuring a seamless experience across various devices. CSS is organized using the BEM methodology, which promotes scalability and maintainability. JavaScript enhances the site's interactivity, such as managing the mobile menu toggle and other user interface interactions.
